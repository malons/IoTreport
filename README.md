# Internet de las Cosas

## Descripción
Simple HTML, formato mínimo para copiar pegar.
Guía para realizar un briefing de empresa sobre internet de las cosas, realizado por un conocido chat de inteligencia artificial al que se le han proporcionado dos o tres pasos de datos mínimos directamente en cada idioma, sin recurrir a traducción.
Al menos en el campo de la Mercadotécnia y la Comunicación, ya no nos planteamos si es conveniente o no la segmentación masiva de nuestro público objetivo confiando la plataforma comunicativa a una solución  IA: EL desafío es detectar y corregir campañas producidas por la IA en nuestros mercados y en competencia con nuestro producto, actividad u oferta.
A notar las sutiles diferencias de conceptos entre las cuánto guías.
